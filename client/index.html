RegExpress

Demystifying regular expressions

Welcome to RegExpress, a tool that lets you build regexes by creating a syntax diagram instead of having to memorize countless esoteric symbols.

RegExpress will show you the text and the "regex railroad" diagram for any regular expression. Type in a regex to see the diagram it creates, or build your own out of the library components.
